# Bedrock Descent

A blocky browser game prototype where you dig and descend through layers to reach bedrock.

## Run locally

```bash
npm install
npm run serve
```

Open <http://localhost:4173>.

## Controls

- `A` / `D` or arrow keys: move
- `W` / `Space` / up arrow: jump
- Click `Start Descent` to begin

Current objective: reach the bedrock layer.
